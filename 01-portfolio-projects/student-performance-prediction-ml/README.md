# Student Performance ML

Proyecto de Machine Learning para predecir el rendimiento de estudiantes usando datos tabulares.

## Objetivo

Construir un pipeline básico de Machine Learning que permita:

1. Cargar datos desde CSV.
2. Limpiar y preparar los datos.
3. Entrenar un modelo con Scikit-learn.
4. Evaluar el modelo con métricas.
5. Guardar el modelo entrenado.
6. Cargar el modelo para hacer predicciones nuevas.
